# react-snowf
React snowflakes component, let it snow on your page! ‚ùÑ

## Install
```bash
npm install react-snowf --save
```

## Usage
```jsx
import Snowf from 'react-snowf';

// Insert component in your code
<Snowf
  amount={50}
  size={5}
  speed={1.5}
  wind={0}
  color="#fff"
  opacity={0.8}
  swing={true}
  swingOffset={1}
  image=""
  zIndex={0}
/>
```
